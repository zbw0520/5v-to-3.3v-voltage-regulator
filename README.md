# 5v-to-3.3v-voltage-regulator
A voltage regulator implemented using ams1117-3.3v on a PCB board with an regulated output of 3.3v.

## What is included in the PCB?

- ams1117-3.3v chip
- 2 leds
- 2 resistors
- 2 ports
- 2 electrolytic capacitors
